# Paper Reader

ArXivやPDFファイルをAIが日本語でわかりやすく解説してくれる、対話型論文読解アシスタントです。

## 起動方法

```
起動.bat をダブルクリック
```

## 何ができるか

| 機能 | 詳細 |
|------|------|
| **論文読み込み** | arXivのURL or PDFファイルをアップロード |
| **セクション自動検出** | Claude APIがタイトル・章構成を自動解析 |
| **先生が解説** | セクションごとに日本語でわかりやすく説明（ストリーミング） |
| **Q&A** | わからなければその場で質問できる |
| **次へ進む** | 「次のセクション▶」ボタン or チャットで「次へ」と入力 |

## 画面構成

```
┌─────────────────────────────────────────────┐
│ サイドバー          │  メインエリア           │
│  - APIキー入力      │  セクション名・進捗     │
│  - URL/ファイル読込 │  ─────────────────     │
│  - 目次ナビゲーション│  先生の解説（チャット）│
│  - リセット         │  質問入力欄             │
└─────────────────────────────────────────────┘
```

## 使い方

1. `起動.bat` をダブルクリックしてブラウザを開く
2. サイドバーに Anthropic APIキーを入力
3. arXivのPDF URL（例: `https://arxiv.org/pdf/1706.03762.pdf`）を貼るか、PDFファイルをアップロード
4. 先生が自動でセクションごとに解説を開始
5. 質問があればチャット入力欄から質問
6. 理解できたら「次のセクション▶」ボタンまたは「次へ」と入力して先へ進む

## 技術スタック

- **Streamlit** — UI
- **PyMuPDF (fitz)** — PDF読み込み・テキスト抽出
- **Anthropic SDK** — `claude-sonnet-4-6` でセクション解析・解説・Q&A
